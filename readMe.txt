HOW TO INSTALL AND RUN THE PROJECT (ReadMeXX.txt)

1. Download the project files as a ZIP (SourceXX.zip) and extract them into a folder. 
   (Alternatively, you can clone from the GitHub repo: 
   https://github.com/hasntezcan/HotelsRezervationSystem.git)

2. Open a terminal and navigate into the project folder:
   cd <folder-name>

3. Make sure Node.js is installed on your system (nodejs.org).

4. Run this command to install all required packages:
   npm install

5. Start the development server:
   npm run dev

6. Open your browser and go to the link shown in the terminal.
